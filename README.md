# Python-system-monitor
This script displays live system metrics in the terminal and logs system statistics.

## Tech Stack
Python 3
psutil
rich

The project uses the Python library psutil to retrieve system metrics and rich to render a clean terminal dashboard.

## Installation
Clone the repository:
git clone https://github.com/Ezraa1122/system-monitor

Navigate to the project folder:
cd system-monitor

Install dependencies:
pip install -r requirements.txt

## Usage
Run the system monitor:
python monitor.py

The dashboard updates every 2 seconds and logs system metrics to:
system.log
